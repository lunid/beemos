IDp wrappers initializer

initialize() 

The main job of wrappers initializer is to performs (depend on the IDp api client it self): - include some libs nedded by this provider, - check IDp key and secret, - set some needed parameters (stored in $this->params) by this IDp api client - create and setup an instance of the IDp api client on $this->api
