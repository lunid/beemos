CARGO/FUNÇÃO

Ao efetuar um novo cadastro de colaborador (consumidor de créditos) é necessário vinculá-lo com uma função. A função define a quantidade de créditos a cada recarga e o total permitido de recargas automáticas (Recarga/Auto). Legenda das colunas:


Cargo/Função: Descrição que será usada no momento do cadastramento do usuário (consumidor de créditos).


Código: Opcional. Preencha-o caso queira carregar um arquivo externo para cadastro automático de muitos usuários.


Lim/Créd.: número de créditos atribuídos a cada recarga do usuário.


Recarga/Auto: número de recargas automáticas permitidas.


Usuários: mostra o número de usuários vinculados ao cargo/função atual.
