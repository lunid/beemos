Package: Default
Hybrid_Provider_Adapter is the basic class which Hybrid_Auth will use to connect users to a given provider.

Basically Hybrid_Provider_Adapterwill create a bridge from your php application to the provider api.

Hybrid_Auth will automatically load Hybrid_Provider_Adapter and create an instance of it for each authenticated provider.
