Package: Auth
Storage driver for fetching login data from a database using ADOdb-PHP.

This storage driver can use all databases which are supported by the ADBdb DB abstraction layer to fetch login data. See http://php.weblogs.com/adodb for information on ADOdb. NOTE: The ADOdb directory MUST be in your PHP include_path!
